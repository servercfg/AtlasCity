TokoVOIP
TokoVOIP er vores in-game snakke system ved hjælp af et specialfremstillet TS-plugin.

Det giver os meget flere muligheder, som i øjeblikket tillader sådanne ting som:
  • Bedre stemmekvalitet
  • Radio system
  • Telefonopkald

TokoVOIP skal være installeret og køre for at spille på vores servere.
