Bilerne bliver ikke slettet hurtigt nok under visning? 
Dette kan skabes af fx. import biler, custom biler, eller langsomme systemer, server eller pc.

Dette kan gøres for at hjælpe det på vej:

Linje 64+65 i GUI:
exports['ac_progressbar']:startUI(500, "Klargører køretøj")
Citizen.Wait(500)

Disse 500 timers kan ændres til større tal, dette skal selvfølgelig gøres for hvert sted de står.
Dette gør at tiden den bruger på at "klargører køretøj" bliver højere.
